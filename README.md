🥇 Project Overview
This project presents a comprehensive analysis of the Olympic Games dataset. The goal was to transform raw historical data into an interactive visual story that highlights global sporting trends, dominant nations, and gender participation evolution over time.

🚀 Key Features
KPI Overview: High-level metrics showing total countries, medals, unique athletes, and sports.

Medal Analytics: Detailed breakdown of Gold, Silver, and Bronze medals by country and sport.

Trend Analysis: A dynamic line chart showing the growth of athlete participation across different eras.

Gender Insights: Comparative analysis of male and female participation and success rates.

Interactive Treemap: Visualizing medal distribution across various sporting categories.

🛠 Tools & Technologies
Power BI: Data Visualization & Dashboarding.

SQL (SSMS): Data cleaning and complex querying.

Excel/CSV: Initial data sourcing.

DAX: Created custom measures for precise data calculation.
